### Initial load (approx.)

`index` + `vendor` + `i18n` + CSS Ôëł **~270 KB raw / ~88 KB gzip** ÔÇö well under the old 500 KB single-bundle warning.

### Heaviest libraries

1. **Three.js** (~480 KB) ÔÇö already lazy-loaded via `PlaceholderPlantScene` in `PlantPage.tsx`
2. **react-markdown + highlight.js** (~288 KB) ÔÇö loaded with Equipment/Chat routes
3. **React + react-router** (~162 KB) ÔÇö core vendor chunk (expected)

### Optimizations applied

- Route-based `React.lazy()` for AI, Dashboard, Knowledge, Login, Settings, Companies, Plant, Equipment, etc.
- `manualChunks` in `vite.config.ts` (vendor, three, markdown, i18n)
- `PageSkeleton` loading states instead of blank screens
- Vite content-hashed filenames for static asset caching

### Unused / dead code

No critical unused entry points found. `logo.svg` kept as runtime fallback only.

### Further options (future, no capability loss)

- Dynamic import highlight.js themes per language
- Replace Three.js with lighter viewer when 3D matures
- `npm run analyze` generates `dist/bundle-stats.html` visual treemap
